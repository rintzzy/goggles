Why Goggles?

Button is a UI/UX component which enhance the total viewing experience to the user. It doesn't matter what you're designing, a button pretty much exists everywhere.

As a designer, you already have frameworks like Bootstrap, Foundation or UIKit. Although they do have some sort of custom button, but your choice is limited.

As alternative, your first thought would be using Photoshopped or vector images on buttons. But in a world of front-end frameworks and floods of tools, what if we had a nice CSS alternative?

Hence, I created this toolkit. It works as a library and extends your choice of designs. The toolkit has several design schemes that suits for your specific project needs. And what's more important that it can be used as Bootstrap theme, for buttons(only). I hope you may find it helpful.


The name is Goggles, because a goggle, being in various colors, it redefines a person's looks - just as buttons do in the realm of web.

Installation & Usage

Step 1:

Just include the stylesheet(.CSS) at the <head> of your project.

/* include via <link> tag */
<link rel="stylesheet" href="path_to_goggles/goggles.css" />
      /* ------OR------ */
/* use the @import keyword in your main stylesheet */
@import url("path_to_goggles/goggles.css");

Step 2:

Add the button class of the desired scheme you want. For the documentation of classes, please refer to the documentation section below.

Documentation

Glad that you chose Goggles!

The namespace

The classnames must include all the required classes for the design scheme you chose. That is, in the class attribute you must specify the core class which is the .btn class. After that you would use the sub-class of the design scheme core, likely, .btn-* and sub-sub-classes, likely, .btn-*-*. It's Bootstrap-like namespace, if you're familiar.

The schemes

Goggles includes 7 types of design schemes :

  • Glossy
  • DuoTone
  • Pattern
  • Metro
  • MetroBG
  • Framed
  • Shadow

Each of these schemes includes several contextual button as found in Bootstrap, plus 3 in the first three schemes.
